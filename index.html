Tutorial Docker Laradock Git no Ubuntu

Instalando, criando um projeto e enviando ao Github

Este simples tutorial ensina o básico para se criar um projeto Laravel utilizando Docker e Laradock, e armazená-lo no Github.

Aqui aprenderá a instalar o Git, Docker e Laradock, Rodar containers para executar os serviços básicos e criar um projeto em Laravel, configurar um banco de dados, conectar-se a ele, realizar um crud e salvar o projeto no Github.
