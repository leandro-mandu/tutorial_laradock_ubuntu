Docker

Setup

Permite a organização dos serviços utilizados no desenvolvimento separados em containers, de forma que se consegue criar um ambiente de desenvolvimento idêntico em diferentes computadores, facilitando a configuração do ambiente de trabalho.

Em nosso projeto utilizamos o Docker para rodar containers, que são como máquinas virtuais, porém mais leves, isto permite isolar os serviços necessários para o desenvolvimento e agilizar as configurações necessárias.

A instalação do Docker no Ubuntu é fácil, e antes é nescessária a instalação do Git, cujo procedimento pode ser acessado pelo menu principal do blog.
